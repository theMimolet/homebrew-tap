# Mimolet's Homebrew Repo
Just adding stuff here, that's useful to me and people I care about

## Luciole font
It's an easy to read font made by specialists in France :
https://luciole-vision.com/

#### Installation

Here's the command to install it :
```sh
brew install --cask theMimolet/tap/font-luciole
```

## .NET SDK (Linux)
This tap provides Linux-only .NET SDK casks.

#### Why?
I added it because there wasn't any Linux support in the official .NET casks, and I wanted to be able to install it on my Linux machine using Homebrew.

#### Installation

Install latest SDK:
```sh
brew install --cask theMimolet/tap/dotnet-sdk-linux
```

Install .NET 8 SDK:
```sh
brew install --cask theMimolet/tap/dotnet-sdk-linux@8
```
